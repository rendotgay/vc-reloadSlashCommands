# Reload Slash Commands
A Vencord plugin for reloading slash commands without reloading the client, for easier testing of applications.

## Features
- Adds a simple reload button
- CTRL + SHIFT + R will also reload

## Installation
1. Follow [this](https://docs.vencord.dev/installing/) if you haven't already.
2. Clone the repository inside your Vencord `src/userplugins` folder (create the `userplugins` folder if it doesn't exist)
```bash
cd Vencord/src/userplugins
git clone https://github.com/rendotgay/vc-typingSounds
pnpm build
````
3. Then reload Vencord and enable the plugin!
